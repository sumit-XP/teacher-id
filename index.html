Sheet Setup

Use the helper file in this folder to connect the form to Google Sheets.

  1. Open sheets.new and create a blank Sheet.
  2. Go to Extensions, then Apps Script.
  3. Paste the code from google-apps-script.gs.
  4. Deploy as a Web App with access set to anyone with the link.
  5. Paste the Web App URL into this form.
